## 数学函数

对数值进行计算，并返回一个数值。数学函数都是标量函数。

```
SELECT ABS(-12);        --返回绝对值（12）
SELECT ACOS(0.5);       --反余弦，取值-1到1之间（1.0471975511966）
SELECT ASIN(0.5);       --反正弦，取值-1到1之间（0.523598775598299）
SELECT ATAN(0.5);       --反正切，取值-1到1之间（0.463647609000806）
SELECT ATN2(0.5, 0.5);  --返回两个坐标指定的点到原点 和 X轴到原点之间的角度。（0.785398163397448）
SELECT CEILING(12.3);   --向上取整；（13）
SELECT COS(30);         --余弦（0.154251449887584）
SELECT COT(30);         --余切（-0.156119952161659）
SELECT DEGREES(3.14);   --输入弧度，返回角度（179.908747671078490000）
SELECT EXP(33.33);      --返回指数值（298562378461581）
SELECT SIGN(-3434);     --返回指定表达式的正（1）、负（-1）、零（0）（-1）
SELECT SIN(60);         --正弦（-0.304810621102217）
SELECT SQRT(4);         --返回平方根（2）
SELECT SQUARE(4);       --返回平方（16）
SELECT TAN(30);         --正切（-6.40533119664628）
```
