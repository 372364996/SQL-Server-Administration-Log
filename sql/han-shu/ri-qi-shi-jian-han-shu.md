## 日期时间函数

对日期和时间执行操作，日期时间函数是标量函数。

用来获取系统日期和时间值的函数：

```
SELECT SYSDATETIME();        --返回包含系统日期和时间的datetime2值（2017-11-03 09:35:57.8236814）
SELECT SYSDATETIMEOFFSET();  --返回包含系统日期和时间的datetimeoffset值（2017-11-03 09:36:50.5447126 +08:00）
SELECT SYSUTCDATETIME();     --返回包含系统日期和时间的datetime2值，日期和时间作为UTC时间返回（2017-11-03 01:38:01.1152090）
SELECT CURRENT_TIMESTAMP;    --返回系统当前的日期和时间的datetime2值（2017-11-03 09:40:53.130）
SELECT GETDATE();            --返回系统日期和时间的datetime2值（2017-11-03 09:41:09.557）
SELECT GETUTCDATE();         --返回系统日期和时间的UTC值（2017-11-03 01:42:19.200）
```

用来获取日期和时间部分的函数：

```
SELECT DATENAME(YYYY, '2017-11-03 01:42:19.200');    --返回日期字符串中指定的日期部分[年YYYY;季度QQ;月MM...]（2017）
SELECT DATEPART(YYYY, '2017-11-03 01:42:19.200');    --返回日期字符串中指定的日期部分[年YYYY;季度QQ;月MM...]（2017）
SELECT DAY('2017-11-03 01:42:19.200');               --返回几号（3）
SELECT MONTH('2017-11-03 01:42:19.200');             --返回月份（11）
SELECT YEAR('2017-11-03 01:42:19.200');              --返回年份（2017）
```

用来修改日期和时间值的函数：

```
SELECT DATEADD(MM,1,'2017-11-03 01:42:19.200');		--将指定的数字1与日期2017-11-03 01:42:19.200根据指定的部分MM（月）相加后返回（2017-12-03 01:42:19.200）
SELECT SWITCHOFFSET('2017-11-03 09:56:44.8559429 +08:00','+06:00');	--更改时区，但保留UTC时间（2017-11-03 07:56:44.8559429 +06:00）
SELECT TODATETIMEOFFSET('2017-11-03 01:42:19.200','+06:00');		--为datetime2值增加时区（2017-11-03 01:42:19.2000000 +06:00）

```
